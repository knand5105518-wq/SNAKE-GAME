# Snake Game
A simple and fun Snake Game built using HTML,CSS,and Java-script.
## Feature-
-Smooth Snake Movement
-Food generation
-Score tracking
-Timer
-High score record
-Game over detection
-Restart Functionally
## Technology
HTML
CSS
Java-script
